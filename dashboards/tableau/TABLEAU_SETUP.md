# Tableau Dashboard Setup Guide

## A-SOC Threat Intelligence Dashboard for Tableau

This guide provides instructions for creating the Tableau dashboard using the output files generated by the A-SOC pipeline.

---

## Required Data Files

Connect Tableau to these CSV files from the `outputs/` directory:

1. **risk_scores.csv** - PIN-level risk data (primary)
2. **state_summary.csv** - State-level aggregations
3. **district_summary.csv** - District-level aggregations
4. **ioc_catalogue.csv** - Detected IOCs
5. **alerts.csv** - Active alerts
6. **daily_summary.csv** - Time-series data

---

## Dashboard Sheets to Create

### Sheet 1: National Threat Heatmap

**Data Source**: state_summary.csv

**Chart Type**: Filled Map (using Tableau's built-in India map)

**Configuration**:
- Geographic Role: Set `state` as State/Province
- Color: avg_risk_score (continuous, Red-Yellow-Green reversed)
- Size: high_risk_pins
- Tooltip: state, avg_risk_score, max_risk_score, pin_count, high_risk_pins

**Color Scale**:
- 0-4: Green (#2EA043)
- 4-6: Yellow (#DBAB09)
- 6-8: Orange (#F85149)
- 8-10: Red (#D73A49)

---

### Sheet 2: Threat Timeline

**Data Source**: daily_summary.csv

**Chart Type**: Line Chart with Dual Axis

**Configuration**:
- X-axis: date (continuous)
- Y-axis 1: total_enrollments (line, blue)
- Y-axis 2: total_demo_updates + total_bio_updates (line, orange)
- Add reference line for average

**Enhancements**:
- Add 7-day moving average trend line
- Highlight dates with extreme values

---

### Sheet 3: Enrollment vs Update Scatter

**Data Source**: risk_scores.csv

**Chart Type**: Scatter Plot

**Configuration**:
- X-axis: total_enrollments
- Y-axis: total_demo_updates + total_bio_updates
- Color: risk_level (categorical)
- Size: risk_score
- Detail: pincode, state, district

**Color Mapping**:
- CRITICAL: #D73A49
- HIGH: #F85149
- MEDIUM: #DBAB09
- LOW: #2EA043

---

### Sheet 4: Demographic & Biometric Anomaly Monitor

**Data Source**: risk_scores.csv

**Chart Type**: Dual Bar Chart

**Configuration**:
- Rows: state (top 15 by risk)
- Columns: 
  - Bar 1: child_ratio_zscore (capped at ±5)
  - Bar 2: bio_recapture_ratio_zscore (capped at ±5)
- Color: Based on Z-score value (diverging)

**Reference Lines**:
- Add reference lines at Z = -3 and Z = +3
- Highlight bars exceeding thresholds

---

### Sheet 5: SOC Alert Queue

**Data Source**: alerts.csv or ioc_catalogue.csv

**Chart Type**: Table

**Columns to Display**:
1. ioc_id / alert_id
2. pattern_name
3. pincode
4. state
5. district
6. risk_score
7. risk_level
8. date_detected
9. recommended_action (truncated)

**Formatting**:
- Conditional formatting on risk_level column
- Sort by risk_score descending
- Enable row-level drill-down

---

## Required Filters

Add these filters to the dashboard:

1. **Date Range** (if date available)
   - Type: Relative Date or Date Range
   - Apply to: All sheets

2. **State**
   - Type: Multi-select dropdown
   - Apply to: All sheets
   - Show "All" option

3. **District**
   - Type: Multi-select dropdown
   - Dependent on State filter
   - Apply to: All sheets

4. **Risk Level**
   - Type: Multi-select
   - Options: CRITICAL, HIGH, MEDIUM, LOW
   - Default: CRITICAL, HIGH selected
   - Apply to: All sheets

## Calculated Fields

Create these calculated fields in Tableau:

### 1. Risk Level Color
```
IF [risk_level] = "CRITICAL" THEN "#D73A49"
ELSEIF [risk_level] = "HIGH" THEN "#F85149"
ELSEIF [risk_level] = "MEDIUM" THEN "#DBAB09"
ELSE "#2EA043"
END
```

### 2. Total Activity
```
[total_enrollments] + [total_demo_updates] + [total_bio_updates]
```

### 3. Update to Enrollment Ratio
```
IF [total_enrollments] > 0 THEN
    ([total_demo_updates] + [total_bio_updates]) / [total_enrollments]
ELSE 0
END
```

### 4. Is High Risk
```
[risk_level] IN ("CRITICAL", "HIGH")
```

### 5. Clipped Z-Score
```
IF [child_ratio_zscore] > 5 THEN 5
ELSEIF [child_ratio_zscore] < -5 THEN -5
ELSE [child_ratio_zscore]
END
```

## Data Refresh

To refresh the dashboard with new data:

1. Run the Python pipeline: `python src/pipeline.py`
2. In Tableau: Data → Refresh All Extracts
3. Verify date ranges update correctly

---

## Constraints

Per the claude.md specification:

- ✅ Use Tableau built-in maps only
- ❌ No external shapefiles
- ❌ No external data joins
- ✅ All calculations derived from dataset fields
- ✅ Deterministic output (no random sampling)

---

## Export Options

The dashboard should support:

1. PDF export for reports
2. Image export for presentations
3. CSV export from tables for further analysis
